# Yazi Cheatsheet

## Navegación
| Tecla | Acción |
|-------|--------|
| `j` / `↓` | Bajar |
| `k` / `↑` | Subir |
| `l` / `→` | Entrar en directorio / abrir archivo |
| `h` / `←` | Volver al directorio padre |
| `g g` | Ir al principio de la lista |
| `G` | Ir al final de la lista |
| `~` | Ir al directorio home |

## Selección
| Tecla | Acción |
|-------|--------|
| `Space` | Seleccionar/deseleccionar archivo |
| `v` | Modo selección visual |
| `V` | Seleccionar todo |
| `Escape` | Cancelar selección |

## Operaciones de archivos
| Tecla | Acción |
|-------|--------|
| `d` | Eliminar archivo(s) |
| `D` | Eliminar permanentemente |
| `r` | Renombrar |
| `a` | Crear archivo |
| `A` | Crear directorio |
| `y` | Copiar (yank) |
| `x` | Cortar |
| `p` | Pegar |
| `P` | Pegar con sobreescritura |

## Búsqueda y filtros
| Tecla | Acción |
|-------|--------|
| `/` | Buscar en directorio actual |
| `n` | Siguiente resultado de búsqueda |
| `N` | Resultado anterior |
| `f` | Filtrar archivos |

## Tabs y paneles
| Tecla | Acción |
|-------|--------|
| `t` | Nueva pestaña |
| `[` | Pestaña anterior |
| `]` | Pestaña siguiente |
| `1`-`9` | Ir a pestaña N |

## Misc
| Tecla | Acción |
|-------|--------|
| `q` | Salir |
| `Q` | Salir y cambiar al directorio actual en la shell |
| `?` | Ayuda |
| `.` | Mostrar/ocultar archivos ocultos |
| `z` | Saltar a directorio con zoxide |

## Instalación en macOS
```bash
brew install yazi
```

## Ejecutar
```bash
yazi
```
